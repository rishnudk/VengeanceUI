
# Installation

Get started with Vengeance UI in a few simple steps.

## Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (version 16.8 or higher)
- [Tailwind CSS](https://tailwindcss.com/) (version 3.0 or higher)

## Install Dependencies

Run the following command to install the necessary dependencies:

```bash
npm install framer-motion clsx tailwind-merge lucide-react
```

## Utilities

Add a `cn` helper to your utilities file (usually `lib/utils.ts`):

```typescript
import { ClassValue, clsx } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

You are now ready to start using Vengeance UI components!
